# Model-Specific Software
Most VDRs may have a specific type of software to conduct analysis on the data. Those for different VDR models discussed in the [VDR Background](/Background/VDR/Requirements.md) section are:

| Name | Expected Data Format / Type | Software Name | Software Link |
|------|-----------------------------|---------------|---------------|
| Danelec Marine DM100 VDR | Proprietary binary container (audio, radar, NMEA 0183/2000, AIS multiplexed) | Danelec Replay / DM Investigator | https://www.danelec.com/products/dm100-vdr-g2#downloads |
| AMI Marine Float-Free VDR (L3 / L3Harris) | Proprietary binary container (bridge audio, radar, ECDIS, NMEA) | AMI VDR Replay / Investigator | https://amimaritime.com/x-vdr/ |
| JRC JCY-1900 VDR | JRC proprietary VDR format (radar, ECDIS, NMEA, audio) | JRC VDR Player / Investigator | https://www.manualslib.com/manual/3515132/Jrc-Jcy-1900.html#manual |
| IMO-Compliant Float-Free VDR (Generic) | Manufacturer-specific proprietary capsule format | Vendor-specific replay tools (Danelec / NetWave / AMI / Furuno / Consilium) | https://www.furuno.com/en/merchant/vdr/#Software |
| Yacht Devices VDR (YDVR-04N / YDVR-02) | Raw NMEA 2000 CAN frames stored in .DAT | YDVR Converter, ydn2kconvert, CAN tools | https://www.yachtd.com/downloads/ |
| Kelvin Hughes MantaDigital VDR | Proprietary Manta container (radar, ECDIS, audio, NMEA) | Manta Replay / Investigator Suite | https://usermanual.wiki/Kelvin-Hughes/DTX-A1-1182062.pdf |
| Furuno VR-3000 Series | Furuno proprietary VDR format (radar, ECDIS, audio, NMEA) | VR3000 Replay Software | https://support.furuno.co.uk/en_US/frequently-asked-questions-vr-3000/vr-3000-liveplayer-maintenance-viewer-v4-v141 |
| NetWave Float-Free VDR | NetWave proprietary capsule format | NetWave Replay Suite | https://www.seasofsolutions.com/sdm_downloads/netwave-player/ |
| Consilium VDR (S-VDR / FDR) | Consilium proprietary container | Consilium Replay Software | https://static.mackaycomm.com/wp-content/uploads/2021/08/consilium_vdrf1_s-vdrs1_b008.1310.1.pdf |
